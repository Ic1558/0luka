# Decision Infrastructure Capability

## Layer

Core Layer

## Purpose

Provide the decision preview and bounded decision memory infrastructure used by the system.

## Scope

- Decision preview structures
- Decision memory boundaries and storage rules
- Evaluation pipeline for decision outputs

## Non-Goals

- Runtime mutation
- Control-plane execution
- Autonomous remediation

## Interfaces

- Inputs: policy outputs, interpreted system state
- Outputs: decision previews, bounded decision memory artifacts
