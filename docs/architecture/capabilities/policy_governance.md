# Policy Governance Capability

## Layer

Core Layer

## Purpose

Define the policy constraints and governance rules that bound system behavior.

## Scope

- Policy definitions and constraints
- Classification criteria and thresholds
- Allow and deny rules for decision infrastructure

## Non-Goals

- Runtime execution
- Automated remediation
- Hidden policy side effects

## Interfaces

- Inputs: interpreted system state, observability signals
- Outputs: policy constraints and policy decisions used by decision infrastructure

- Canonical owner: this document
- Execution owner: policy services in the core layer
