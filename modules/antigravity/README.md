# Antigravity Module Boundary

Antigravity is a bounded 0luka module pack.

It owns:

- domain intelligence
- realtime detection logic
- market connectors
- module-local docs

It does not own:

- kernel secrets policy
- host runtime supervision
- system survival governance
- observability source-of-truth ownership

Phase A freezes new Antigravity feature growth while the survival layer is
kernelized into 0luka core/runtime/observability.
